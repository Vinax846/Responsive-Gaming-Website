# Responsive-Gaming-Website
A modern and responsive gaming website designed with a sleek dark neon UI. It features tournament schedules, match details, news updates, and interactive sections like join team and newsletter signup. Built using HTML, CSS, and JavaScript to deliver an engaging user experience.


# 🎮 VINAX Gaming Website

A modern and responsive gaming website built to showcase tournaments, matches, and gaming news. Designed with a futuristic UI and smooth user experience.

---

## 🚀 Features

* 🔥 Attractive Gaming UI (Dark + Neon Theme)
* 📱 Fully Responsive Design
* 🏆 Tournament Section with Match Details
* 📰 News & Headlines Section
* 👥 Join Team / Call-To-Action Buttons
* 📧 Newsletter Subscription
* 🔝 Scroll-to-top Button
* 📌 Footer with Contact & Useful Links

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Responsive Design (Flexbox / Grid)
